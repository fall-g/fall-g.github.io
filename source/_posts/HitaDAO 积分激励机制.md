## 摘要

本文档概述了参与者通过在 HitaDAO GitHub 社区中的特定贡献行为获得积分的详细机制。这些行为包括对 HitaDAO 组织的跟踪关注（Follow）以及对项目仓库的星标（Star）。参与者所获得的积分将通过智能合约自动映射至其以太坊地址，确保积分的透明和安全发放。

## 积分获取细则

### 1. 关注（Follow）激励

- **操作定义**：在 GitHub 平台上对 HitaDAO 组织进行关注。
- **积分值**：每完成一次关注操作，参与者将获得 10 积分。
- **资格条件**：积分仅授予成功关注 HitaDAO 组织的参与者。

### 2. 星标（Star）激励

- **操作定义**：在 GitHub 上对 HitaDAO 指定的仓库进行星标。
- **积分值**：每完成一次星标操作，参与者将获得 5 积分。
- **资格条件**：积分仅授予对 HitaDAO 指定仓库进行星标的参与者。

## 积分发放流程

- **发放时间**：积分将在参与者完成操作后的 48 小时内发放。
- **发放方式**：积分将通过智能合约自动映射至参与者的以太坊地址，确保流程的自动化和去中心化。

## 操作指导

### 如何提交以太坊地址

- **步骤 1**：登录至 GitHub 平台并导航至 HitaDAO 组织页面。

- **步骤 2**：进入 hitadao 仓库，点击 "Issues" 选项。

  ![](issues-arrow.png)

- **步骤 3**：在 "Issues" 页面，提交一个新的 issue。

  ![](issue-arrow.png)

- **步骤 4**：在 issue 的标题（title）中输入 `以太坊: 您的昵称`，在描述（description）中输入您的以太坊账户地址。

### 如何关注 HitaDAO

- **步骤 1**：登录至 GitHub 平台并导航至 HitaDAO 组织页面。

- **步骤 2**：在组织页面，点击 "Follow" 按钮以关注组织。

  ![](follow-arrow.png)

### 如何星标 HitaDAO 项目

- **步骤 1**：登录至 GitHub 平台并搜索 HitaDAO。

- **步骤 2**：找到并进入指定的 HitaDAO 项目仓库。

- **步骤 3**：在仓库页面，点击 "Star" 按钮以星标项目。

  ![](star-arrow.png)

## 注意事项

- 请确保您的以太坊地址准确无误，以便积分能够顺利映射至您的账户。
- 积分一旦发放，将无法进行补发或转移，请在操作前仔细核对您的账户信息。
- 本积分激励机制的最终解释权归 HitaDAO 所有。

## 常见问题解答

- **Q: 如何确保我的积分能够及时到账？**
  - A: 请确保您的 GitHub 操作符合积分获取条件，并且您的以太坊地址已正确提交至 issue 中。
- **Q: 如果我在规定时间内未收到积分，该怎么办？**
  - A: 请首先检查您的以太坊地址是否正确，并确认您已按照操作指导正确执行了相关操作。如有疑问，请联系 HitaDAO 客服。
- **Q: 积分是否有使用期限？**
  - A: 积分没有使用期限，您可以在任何时候使用积分。
